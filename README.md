# pmamico's homebrew tap

## `jssl`

`jssl` is a keytool alternative for installing ssl certificates in java keystore.  
See more at https://github.com/pmamico/jssl

### Install

`brew install pmamico/keg/jssl`

Or `brew tap pmamico/keg` and then `brew install jssl`.

### Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).

## `matrix-cam`

`matrix-cam` renders live ASCII camera feeds with Matrix-style glyphs inside the macOS terminal.  
See more at https://github.com/pmamico/matrix-cam

### Install

`brew install pmamico/keg/matrix-cam`

Or `brew tap pmamico/keg` and then `brew install matrix-cam`.
